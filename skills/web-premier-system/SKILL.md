# web-premier-system

## Propósito

Definir el estándar de producción para propuestas web premier en Humanio.

## Objetivo

Crear una experiencia más específica, diferenciada y valiosa cuando la oportunidad comercial justifica una inversión mayor de tiempo, criterio y tokens.

## Cuándo usar

Usar este sistema solo cuando `delivery_mode = premier`.

Esto aplica típicamente a:
- leads inbound
- prospectos que solicitaron información
- prospectos que solicitaron demo
- leads cargados directamente por dirección
- oportunidades marcadas como urgentes por el CEO
- casos de alto valor o alta probabilidad comercial

## Filosofía

El modo premier no existe para “hacer algo más bonito”.
Existe para elevar percepción de valor donde sí hay señal de retorno.

La personalización adicional debe responder al negocio, no al ego creativo.

## Regla principal

Aquí sí puedes aumentar especificidad visual, estructura narrativa y sofisticación de interacción.
Pero cada capa extra debe mejorar:
- claridad comercial
- percepción de valor
- adecuación al tipo de negocio
- efectividad de la propuesta

## Componentes obligatorios

Siempre deben existir:
- `/index.html`
- `/propuesta/index.html`
- `/reporte/index.html`

## Libertades permitidas

En modo premier sí puedes variar:
- layout
- dirección visual
- hero
- jerarquía de secciones
- ritmo narrativo
- interacción visual
- riqueza del sistema tipográfico
- uso de imagen y composición

## Límites

No hagas esto:
- añadir complejidad solo por lucimiento
- usar efectos excesivos sin función
- parecer agencia genérica
- prometer resultados no sustentados
- romper claridad por querer impresionar

## Estándar esperado

Un output premier debe:
- sentirse más específico que template
- tener una narrativa visual más intencional
- reflejar mejor el tipo de negocio
- justificar el mayor esfuerzo invertido
- elevar la percepción de profesionalismo y valor

## Nivel de interacción

El `interaction_profile` puede elevarse más que en template, pero debe seguir siendo coherente con:
- el giro del prospecto
- el tono recomendado
- la claridad del sitio
- la velocidad de carga razonable

## Regla de diferenciación

En modo premier, evita repetir combinaciones visuales recientes cuando sea posible.
La repetición exacta debe ser menos frecuente que en template.

## Relación con DesignPlanner

WebBuilder debe ejecutar lo definido por `DESIGN_SPEC`.
No debe reinventar la dirección creativa por su cuenta.

## Identidad de marca

Humanio es una consultora de Inteligencia Artificial para negocios.

Nunca presentes el trabajo como simple diseño web aislado.
La web es una puerta de entrada a una propuesta mayor de automatización, agentes y sistemas de IA.

Firma correcta:
"Humanio — Inteligencia Artificial para negocios"
